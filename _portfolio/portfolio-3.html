This work explores the QA augmentation for events in news articles. Our altimate goal is to automatically extract structured events from unlabeled unstructured articles. In this work, we focus on a subtask: extract QA pairs for arguments given an event. The synthetic data severs as augmented data to train the QA event extraction model.
